# ZeroTrashService 

ZeroTrashService is a .NET-based background service that clears the Recycle Bin (C:$Recycle.Bin) on the C: drive in Windows.

---

## Warning

* Once started, this service permanently deletes all files in the Recycle Bin on the C: drive. This operation is irreversible; please ensure you do not have any critical files in the Recycle Bin before running this service.
